# DNS em detalhes

## Descrição

O DNS (Domain Name System) é responsável por traduzir nomes de domínio em endereços IP.

Por exemplo, quando um usuário acessa um site, o DNS converte o nome do domínio para o endereço IP do servidor.

## Conceitos aprendidos

- funcionamento do DNS
- resolução de nomes.
- tipos de registros DNS

## Tipos de registros DNS

A Record  
Relaciona um domínio a um endereço IPv4.

AAAA Record  
Relaciona um domínio a um endereço IPv6.

MX Record  
Define servidores responsáveis por e-mails.

CNAME Record  
Cria um alias para outro domínio.

## Aplicação em cibersegurança

O DNS é frequentemente analisado em investigações de segurança para identificar:

- domínios maliciosos
- infraestrutura de ataques
- atividades suspeitas

## Conclusão

O DNS é uma tecnologia essencial para o funcionamento da internet e para análises de segurança.
