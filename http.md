# HTTP em detalhes

## Descrição

O HTTP (Hypertext Transfer Protocol) é um protocolo utilizado para comunicação entre navegadores e servidores web.

Ele permite que usuários solicitem páginas e recursos de um servidor.

## Conceitos aprendidos

- requisições HTTP
- respostas do servidor
- métodos HTTP

## Métodos HTTP comuns

GET  
Solicita informações de um servidor.

POST  
Envia dados para um servidor.

PUT  
Atualiza informações existentes.

DELETE  
Remove recursos.

## Aplicação em cibersegurança

Profissionais de segurança analisam requisições HTTP para identificar:

- ataques web
- tentativas de exploração
- tráfego malicioso

## Conclusão

Compreender o funcionamento do HTTP é essencial para análise de aplicações web e identificação de vulnerabilidades.
