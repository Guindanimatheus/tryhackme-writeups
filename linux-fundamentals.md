# Linux Fundamentals 

## Descrição

Esta sala apresenta os fundamentos do sistema operacional Linux e ensina como utilizar o terminal para navegar e interagir com o sistema.

## Objetivos da sala

- Entender o funcionamento básico do Linux
- Aprender comandos essenciais do terminal
- Navegar pelo sistema de arquivos

## Comandos aprendidos

ls  
Lista arquivos e diretórios.

cd  
Permite navegar entre diretórios.

pwd  
Mostra o diretório atual.

cat  
Exibe o conteúdo de um arquivo.

mkdir  
Cria novos diretórios.

rm  
Remove arquivos ou diretórios.

## Aplicação na Cibersegurança

O conhecimento de Linux é essencial para profissionais de segurança, pois muitos servidores e ferramentas de segurança utilizam sistemas Linux.

Durante atividades de pentest ou análise de sistemas, o uso do terminal é fundamental para navegar pelo sistema, analisar arquivos e executar ferramentas.

## Conclusão

Esta sala fornece uma base importante para o uso de Linux, que é uma habilidade essencial para profissionais de segurança da informação.
