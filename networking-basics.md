# Fundamentos de Redes

## Descrição

Esta sala apresenta os conceitos básicos de redes de computadores e como dispositivos se comunicam através da internet.

## Conceitos aprendidos

- O que é uma rede de computadores
- Comunicação entre dispositivos
- Endereçamento IP
- Estrutura básica de redes

## Componentes de rede

Roteadores  
Responsáveis por encaminhar dados entre redes.

Switches  
Conectam dispositivos dentro de uma rede local.

Dispositivos finais  
Computadores, servidores e outros equipamentos conectados à rede.

## Aplicação em cibersegurança

Entender redes é essencial para identificar:

- tráfego suspeito
- possíveis ataques
- comportamento anômalo na rede

## Conclusão

Redes de computadores são a base da comunicação digital e um conhecimento essencial para profissionais de segurança.
