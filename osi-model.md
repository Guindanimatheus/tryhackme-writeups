# Modelo OSI

## Descrição

O modelo OSI é um modelo conceitual que descreve como os dados são transmitidos através de uma rede em diferentes camadas.

## As 7 camadas do modelo OSI

1. Física
2. Enlace de dados
3. Rede
4. Transporte
5. Sessão
6. Apresentação.
7. Aplicação

Cada camada possui uma função específica no processo de comunicação entre dispositivos.

## Aplicação em cibersegurança

O modelo OSI ajuda profissionais de segurança a entender:

- onde ocorrem falhas de comunicação
- em qual camada um ataque pode acontecer
- como analisar tráfego de rede

## Conclusão

O modelo OSI é fundamental para compreender o funcionamento das redes e analisar problemas de comunicação.
