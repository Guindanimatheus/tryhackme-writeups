# Fundamentos do Windows

## Descrição

Esta sala apresenta os conceitos básicos do sistema operacional Windows, incluindo a interface do sistema, gerenciamento de arquivos e configurações importantes do sistema.

## Conceitos aprendidos

- Estrutura do sistema Windows
- Sistema de arquivos NTFS
- Painel de Controle
- Controle de Conta de Usuário (UAC)

## Ferramentas importantes

Explorador de arquivos  
Utilizado para navegar e gerenciar arquivos.

Painel de Controle  
Permite acessar configurações do sistema.

Gerenciamento de usuários  
Responsável pelo controle de contas e permissões.

## Aplicação em cibersegurança

O Windows é um dos sistemas mais utilizados em ambientes corporativos.

Profissionais de segurança precisam entender:

- permissões de usuários
- estrutura de arquivos
- configurações de segurança

## Conclusão

Compreender o funcionamento do Windows é essencial para identificar vulnerabilidades e analisar sistemas em ambientes corporativos.
